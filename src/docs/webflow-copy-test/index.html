Test — presse-papiers Webflow (Xscp)

Vérifie que webflow-clipboard écrit bien application/json + text/plain pour un collage dans le Designer. Le flux fiable est : copier dans Webflow → coller ici → recopier → coller dans Webflow.

Étape A — Coller depuis le Designer

  1. Ouvrez un site dans le Webflow Designer.
  2. Sélectionnez un élément (ex. une div) et Ctrl/Cmd+C.
  3. Cliquez dans la zone ci-dessous et Ctrl/Cmd+V (le JSON doit apparaître).
