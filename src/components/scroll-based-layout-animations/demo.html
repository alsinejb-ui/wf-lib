Détail

Reproduction du comportement « On-Scroll Image Layout Animations » : GSAP Flip + ScrollTrigger (pin + scrub), option Lenis. Remplacez les images par les vôtres.

Chaque bloc .gallery peut avoir data-csla-preset="1"…"9" ou data-csla-options (JSON).
